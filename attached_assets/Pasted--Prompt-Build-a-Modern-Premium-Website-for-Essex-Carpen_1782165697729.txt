# Prompt: Build a Modern Premium Website for Essex Carpenters

Create a modern, high-end, responsive website for a carpentry and property improvement company called **Essex Carpenters**.

## Design Style

* Premium construction and tradesman aesthetic.
* Dark charcoal (#1A1A1A), white (#FFFFFF), and warm wood accent colours (#C89B5E).
* Clean and spacious layout with smooth animations.
* Use glassmorphism and subtle shadows.
* Modern typography using **Poppins** or **Inter**.
* Rounded cards and soft hover animations.
* Mobile-first responsive design.
* Include dark and light mode support.

## Technology

* React + Next.js.
* Tailwind CSS.
* Framer Motion for animations.
* Three.js or React Three Fiber for interactive 3D elements.
* Optimised SEO and performance.
